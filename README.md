## :wrench: Instalação e Uso

```bash
Ver scripts no package.json
```
<br>

### `yarn start`

Rode a aplicação em modo de desenvolvimento:

Abra o link: [http://localhost:3000](http://localhost:3000) para ver no navegador.

### `yarn test`

Para rodar os testes.
Veja mais sobre testes no link: [running tests](https://facebook.github.io/create-react-app/docs/running-tests).

### `yarn build`

Para gerar o build da aplicação.

## Obs:

Rodar apenas isso que é o que falta para o meu template tailwind+react

```
yarn add node-sass -D
yarn add postcss-cli autoprefixer -D
```

Editar o package.json
```
"build:css": "postcss src/styles/index.scss -o src/styles/tailwind.output.scss "
```
